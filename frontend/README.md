## 起動方法

```bash
yarn dev
```

## ページ

| path                 | note                             | ページ名      |
|:---------------------|:---------------------------------|:----------|
| /add                 | 金額を登録                            | 入出金登録     |
| /balance             | 今月使った金額が表示される                    | 残高確認      |
| /balance/{yyyy}/{mm} | その月使った金額が表示される                   | 残高確認      |
| /users               | 金額の管理をするユーザを表示                   | 対象ユーザ一覧   |
| /estimate            | 今後の出費額や必要な貯金額などを表示               | 予測        |
| /calendar            | 今後発生するお金の一覧を表示。貯めなければならない金額など表示。 | 出費一覧      |
